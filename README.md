## Installation

```bash
git clone <this repo>
npm install
```

## Getting Started

1. Modify `config.yaml` with your Personal access `githubToken` with `public_repo` access.
2. Adjust `repos` list or `repoSearch` in the `config.yaml`
3. Run `npm run apply`
